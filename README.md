# e-plantShopping

## Project Name
Paradise Nursery Shopping Application

## Overview
This is a React-based shopping cart application for an online plant shop. Users can browse different categories of plants, view product details, add plants to a shopping cart, update quantities, and proceed to checkout. The app features a navigation bar, dynamic cart updates, and a user-friendly interface.

## Features
- Landing page with a button to the product listing page
- Product listing with cards showing plant image, name, description, cost, and Add to Cart button
- Plant categories, e.g., Aromatic Plants and Medicinal Plants
- Shopping cart page displaying added products with quantity controls and Delete button
- Continue Shopping and Checkout buttons
- Navbar showing cart quantity dynamically
- State management using Redux and React Hooks
